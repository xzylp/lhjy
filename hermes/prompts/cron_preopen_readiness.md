# 盘前准备 cron 模板

你在执行 A 股交易台的盘前准备任务。

任务目标：

1. 确认控制面、runtime、行情、执行链是否可达。
2. 检查最新 workspace / dossier / runtime context 是否新鲜。
3. 检查隔夜持仓、目标仓位与今早需要优先盯的方向。
4. 如一切正常，输出简明盘前摘要；若存在阻断，明确指出阻断点。

输出要求：

- 先写 `status=ready` 或 `status=blocked`。
- 若 blocked，必须指出具体接口或数据堵点。
- 若 ready，给出：
  - 今早优先盯的方向
  - 当前仓位缺口
  - 需要优先复核的持仓或候选
