# ashare-system-v2 升级需求文档

> 基于龙虾量化全景架构，结合 v1 已有基础，规划 v2 升级路线。

---

## 一、现状与差距总览

| 模块 | v1 现状 | v2 目标 | 差距 |
|------|--------|--------|------|
| 策略体系 | 5层漏斗 + 多因子评分 | 14+策略融合 + 黄金策略组合 | 需扩展策略库 |
| AI模型体系 | 无 ML 模型，仅规则评分 | 7大 AI 模型全覆盖 | **全新建设** |
| 因子引擎 | ~10个硬编码指标 | 500+因子引擎 | **全新建设** |
| 实时监控 | Watchdog + 健康检查 | 12大监控模块 | 大幅扩展 |
| 情绪周期 | 无 | 四阶段框架 + 拐点识别 | **全新建设** |
| 数据系统 | AkShare 基础接入 | 7类数据 + 清洗/修复/验证 | 需体系化 |
| 回测系统 | 无 | 无限迭代回测 + 组合回测 | **全新建设** |
| 交易系统 | QMT下单 + 风控拦截 | 选股/买入/卖出/做T/仓位全流程 | 需深化 |
| 风控体系 | 10项规则 | 6大维度 + 情绪保护 | 需升级 |
| 报告系统 | 审计日志 | 实时/日终/策略报告 + 飞书推送 | 需体系化 |
| 通信系统 | 无 | 飞书推送 + 告警通知 | **全新建设** |
| 学习系统 | 无 | 持续学习 + 策略自进化 | **全新建设** |

## 二、十二大子系统需求明细

### 1. 策略体系 (目标完成度: 90%)

**v1 已有:** 5层选股漏斗 (宏观→板块→技术→事件→风控)

**v2 新增需求:**

| 需求 | 说明 | 优先级 |
|------|------|--------|
| 多策略注册表 | 支持策略插件化注册、动态启停 | P0 |
| 黄金策略组合 | 多策略加权融合，输出综合信号 | P0 |
| 多周期共振 | 日线/60分/15分多周期信号共振确认 | P1 |
| 板块轮动策略 | 基于板块强度排名的轮动择时 | P1 |
| 游资跟踪策略 | 龙虎榜席位关联分析 + 跟随信号 | P1 |
| 策略优化报告 | 每轮回测后自动生成策略优化建议 | P2 |

**综合胜率目标:** 72%+ (加权平均)

---

### 2. AI 模型体系 (目标完成度: 85%)

**v1 已有:** 无 ML 模型，纯规则评分

**v2 新增需求:**

| 模型 | 用途 | 目标指标 | 优先级 |
|------|------|----------|--------|
| XGBoost 选股模型 | 核心选股评分 | AUC > 0.84 | P0 |
| Transformer 序列模型 | 板块轮动 + 序列预测 | AUC > 0.81 | P0 |
| LSTM 时序模型 | 趋势预测 (带 Attention) | 待定 | P1 |
| NLP 情感模型 | 新闻/公告情绪打分 | 准确率 > 75% | P1 |
| 高频盘口模型 | 盘口微观信号识别 | 准确率 > 59% | P2 |
| PPO 强化学习 | 仓位动态优化 | 年化收益 > 22% | P2 |
| 模型集成 (Ensemble) | 多模型融合决策 | 准确率 > 76% | P0 |

**配套需求:**
- 统一的模型训练 / 推理 / 版本管理框架
- 模型 AUC/准确率自动评估 pipeline
- 支持离线训练 + 在线推理分离部署

---

### 3. 因子引擎体系 (目标完成度: 95%)

**v1 已有:** ~10 个硬编码技术指标 (MA/MACD/RSI 等)

**v2 新增需求:**

| 因子类别 | 目标数量 | 说明 | 优先级 |
|----------|----------|------|--------|
| 基础因子 | 150+ | 量价、技术指标、财务比率 | P0 |
| 微观结构因子 | 80+ | 盘口深度、委比、成交分布 | P1 |
| 另类数据因子 | 70+ | 舆情、搜索热度、社交媒体 | P2 |
| 产业链因子 | 100+ | 上下游关联、供需关系 | P2 |
| 行为金融因子 | 100+ | 动量反转、过度反应、羊群效应 | P1 |
| 宏观因子 | 50+ | 利率、汇率、PMI、社融 | P1 |

**总计: 500+ 因子**

**配套需求:**
- 因子注册表 (声明式定义 + 自动发现)
- 因子批量计算引擎 (支持增量更新)
- 因子 IC/IR 有效性验证
- 因子去极值 (Winsorize) / 标准化 / 中性化 pipeline
- Top-N 核心因子自动筛选 (基于 XGBoost 特征重要性)
